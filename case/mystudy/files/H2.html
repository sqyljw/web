html基本结构

这是一个基本html的结构：
<html>
  <head>
  </head>
  <body>
  </body>
</html>
<!DOCTYPE>不是 HTML 标签。它为浏览器提供一项信息（声明），即 HTML 是用什么版本编写的。 HTML5 <!DOCTYPE html> HTML 4.01 <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd"> XHTML 1.0 <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">

  • body文本标签
    1. <body bgcolor="red">增加背景色的</body>
    2. <body background="images/c7.gif">给网页加背景图片</body>
    3. <body bgproperties="fixed">背景固定，针对ie浏览器有效</body>
    4. <body link="red">链接的默认颜色(还没访问过) </body>
    5. <body alink="red">链接的激活颜色 </body>
    6. <body vlink="red">链接访问过后颜色 </body>
    7. <body topmargin="20">顶部页边距表示</body>
    8. <body bottomargin="20">底部页边距表示</body>
    9. <body leftmargin="20">左部页边距表示</body>
    10. <body rightmargin="20">右部页边距表示</body>
